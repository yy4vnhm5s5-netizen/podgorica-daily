# Project

## Product boundaries

Podgorica Daily is a public information product, not a social network or generic content aggregator. Public information remains useful without an account; accounts serve approved editorial and administrative workflows.

## Delivery standards

- Responsive and accessible behaviour is a baseline.
- Every external datum has ownership, provenance, freshness expectations, and failure behaviour.
- Secrets never enter client bundles, repositories, logs, or issue content.
- Changes ship through reviewed pull requests with automated checks and traceable commits.

## Decisions

Data providers, maps, identity, persistence, hosting, and AI models require an architecture decision record covering licensing, cost, security, privacy, ownership, and exit strategy.
